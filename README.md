# corporate_data_analysis2

## 概要（OverView）
Pythonのデータ分析ライブラリ「Pandas」を使用して、複数企業の財務データを効率的に分析・評価するプログラムです。
手入力によるデータ管理から、表形式（DataFrame）による一括管理へとステップアップし、投資判断に役立つランキング作成や条件検索を自動化しています。

## 使い方（Usage）
1. Pythonをインストールする
2. pandasライブラリをインストールする
3. 以下のプログラムを実行する
   corporate_data_analysis2.py

## 学んだこと（what I laerned）
- DataFrameの作成
  辞書型のデータから、分析に適した行列構造(DataFrame)へ変換する方法
  
- ソート機能
  「ascending=False」を使い分け、昇順（値が低い順）や降順（値が高い順）のランキングを作成する方法
  
- 統計量の算出
  特定の指標を指定し、その平均値を一瞬で算出する数値計算の方法
  
- 複数条件によるフィルタリング
  &(and条件)を使い、複数の基準を同時に満たす企業を抽出するデータ抽出スキル

## 目的（Purpose）
- 大量の企業データの中から、客観的な指標に基づいて「優良企業」を効率的に見つけ出すための自動分析スキルを習得するため。

- データサイエンスの現場で必須となるPandasライブラリの基礎を固め、実務的な財務分析に応用するため。
